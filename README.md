# aisc-csharp

Test project for the Checkmarx AI Supply Chain Security (AISC) scanner.
Checks what AISC detects in C#/.NET. Not meant to run - it's for scanning

## What's inside

- Sdks.cs — OpenAI, Azure OpenAI, and Anthropic client setup
- SemanticKernel.cs — a Semantic Kernel chat example
- Agents.cs — an AutoGen agent
- McpServer.cs — a simple MCP server with a sample tool
- McpClient.cs — an MCP client that calls the server
- AiCsharpSample.csproj — package references
